Detailed Description

Orderer render pass.

Maintains the order in which drawables are added.

Note
This pass by default doesn't do depth testing or writing
Lighting is disabled for all drawables bound to this kind of pass

By default Renderer creates one of these as a AfterPost pass, using RenderPass::DefaultWeight.
